# Whistled Melody Analysis with Librosa

This mini project analyzes a ~19-second whistled melody using Python and Librosa.

The notebook demonstrates:
- Loading and visualizing the waveform
- Computing a mel-spectrogram
- Estimating tempo (≈ 83 BPM)
- Estimating the fundamental frequency (F0) contour with `librosa.yin`
- Mapping F0 values to musical note names and inspecting the pitch distribution (D6 and its neighbors as the most frequent pitches)

My goal is to connect my musical ear (intonation, melodic contour) with concrete audio features and evaluation methods that can be used in ML-based music creation workflows.
